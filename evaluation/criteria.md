# Evaluation Criteria

Use these criteria to assess prompt quality:

1) **Correctness** — output matches ground truth
2) **Format validity** — parseable JSON, no extra text
3) **Completeness** — required fields filled or null
4) **Consistency** — same input → same structure
5) **Safety** — respects constraints and avoids unsafe content
6) **Robustness** — handles ambiguity and edge cases
